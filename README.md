📌 Overview
This project is a Java implementation of the RSA Cryptographic Algorithm used for secure message encryption and decryption. It demonstrates the fundamentals of asymmetric cryptography using public and private keys.

✨ Features
RSA key generation using prime numbers
Encryption of plaintext messages
Decryption of ciphertext messages
Modular arithmetic operations
GCD calculation
Menu-driven interface using switch-case
🛠️ Technologies Used
Java
Object-Oriented Programming
Modular Arithmetic
📚 Concepts Covered
Public Key Cryptography
RSA Algorithm
Euler’s Totient Function
Modular Inverse
Encryption & Decryption
▶️ How to Run
Clone the repository
Open the project in any Java IDE
Compile the program
Run the main Java file

🎯 Purpose
This project was created to understand the practical implementation of cryptography concepts and secure communication systems using Java.
